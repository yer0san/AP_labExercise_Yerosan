## Getting Started

Before u start, initiate the server with
-  javac -cp "../../../lib/mariadb-java-client-3.5.8.jar" shared/*.java university/*.java
-  java -cp ".:../../../lib/mariadb-java-client-3.5.8.jar" university.Server

Then run maven with
-  mvn javafx:run
